# Vue-pandora

A component library of custom secondary encapsulation about element-ui

# Install


  yarn add vue-pandora

# Usage

import VuePandora from 'vue-pandora'
import 'vue-pandora/lib/vuepandora.css'

vue.use(VuePandora)
